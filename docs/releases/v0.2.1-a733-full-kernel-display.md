# v0.2.1 A733 Full Kernel Display

Date: 2026-07-14

This release contains the complete, board-verified A7Z display kernel package.

## Verified Result

- Package: `linux-image-5.15.147-21.1-a733` version `5.15.147-21.1+display2`.
- Boot entry: standard extlinux `l0` with the package-generated initramfs.
- Kernel: `5.15.147-21.1-a733`.
- HDMI: `FLY-HDMI-LCD7` at native `1024x600@60Hz`.
- Network: AIC8800 Wi-Fi and SSH working at `192.168.123.210`.
- Recovery: vendor `5.15.147-21-a733` remains available as `l1`.

## Install

Run the install once and wait for it to finish. The package rebuilds DKMS
modules and initramfs; do not start another `dpkg` or DKMS process.

The recommended path is the release deployment script:

```bash
sudo ./deploy_a733_display_kernel.sh \
  linux-image-5.15.147-21.1-a733_5.15.147-21.1+display2_arm64.deb \
  --activate
sudo reboot
```

The equivalent manual sequence is:

```bash
sudo sed -i 's/^default l0$/default l1/' /boot/extlinux/extlinux.conf
sudo dpkg -i linux-image-5.15.147-21.1-a733_5.15.147-21.1+display2_arm64.deb
sudo dkms status
sudo sed -i 's/^default l1$/default l0/' /boot/extlinux/extlinux.conf
sync
sudo reboot
```

Expected validation:

```bash
uname -r
dpkg -s linux-image-5.15.147-21.1-a733 | grep -E '^(Status|Version):'
sudo journalctl -b -k --no-pager | grep -E 'Configuration mode|drm hdmi mode set'
```

See the bilingual experiment record for failure analysis and offline recovery.
