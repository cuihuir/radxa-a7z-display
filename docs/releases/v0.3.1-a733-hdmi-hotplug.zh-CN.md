# A733 HDMI 热插拔与启动恢复更新

该补丁版本将已验证的 Debian 12 KDE + PowerVR 系统从 `display2`/`gpu7`
更新到 `display3`/`gpu8`。

## 修复内容

- HDMI 拔出并重新插入后可自动恢复，不需要 udev 规则或手动执行 xrandr。
- HDMI HPD 处理函数不再绕过 DRM atomic 状态直接关闭硬件。
- 内核包显式包含 `l0` 启动所需的 A7Z DTB。
- 内核和 GPU 部署工具会在 `u-boot-update` 后恢复 `l1` 的独立 vendor DTB 和
  `module_blacklist=pvrsrvkm`。
- `gpu8` 使用兼容 `display2` 到 `display3` 的内核包依赖范围，不再精确锁定单一
  包修订版本。

## 安装

保留恢复项 `l1`，先安装内核包，再安装 GPU 包：

```bash
sudo tools/deploy_a733_display_kernel.sh \
  linux-image-5.15.147-21.1-a733_5.15.147-21.1+display3_arm64.deb \
  --activate
sudo tools/deploy_a733_gpu.sh \
  a733-pvr-gpu_24.2.6603887+gpu8_arm64.deb \
  --activate
sudo reboot
```

## 验证结果

2026 年 7 月 20 日实机验证通过：

- `l0` 从内核包 `5.15.147-21.1+display3` 启动内核
  `5.15.147-21.1-a733`；
- `pvrsrvkm 24.2.6603887` 正常加载并创建 `card1`/`renderD128`；
- SDDM 在 HDMI-A-1 使用原生 `1024x600@60Hz`；
- 实际拔出并重新插入 HDMI 后显示自动恢复；
- AIC8800 Wi-Fi、SSH、DKMS 模块以及恢复项 `l1` 均保持可用。

本版本提供更新包、部署工具、双语发布说明和 SHA-256 校验，不替换 `v0.3.0`
发布的完整镜像。
