# v0.2.1 A733 完整显示内核

日期：2026-07-14

本版本包含已在 A7Z 实机验证的完整显示内核包。

## 验证结果

- 包：`linux-image-5.15.147-21.1-a733`，版本 `5.15.147-21.1+display2`。
- 启动项：标准 extlinux `l0`，使用包生成的 initramfs。
- 内核：`5.15.147-21.1-a733`。
- HDMI：`FLY-HDMI-LCD7` 原生 `1024x600@60Hz`。
- 网络：AIC8800 Wi-Fi 和 SSH 正常，地址为 `192.168.123.210`。
- 回滚：原厂 `5.15.147-21-a733` 仍保留为 `l1`。

## 安装

安装命令只运行一次并等待完成。包会重建 DKMS 模块和 initramfs，期间不能启动第二个
`dpkg` 或 DKMS 进程。

推荐使用发布中的安全部署脚本：

```bash
sudo ./deploy_a733_display_kernel.sh \
  linux-image-5.15.147-21.1-a733_5.15.147-21.1+display2_arm64.deb \
  --activate
sudo reboot
```

等价的手工步骤为：

```bash
sudo sed -i 's/^default l0$/default l1/' /boot/extlinux/extlinux.conf
sudo dpkg -i linux-image-5.15.147-21.1-a733_5.15.147-21.1+display2_arm64.deb
sudo dkms status
sudo sed -i 's/^default l1$/default l0/' /boot/extlinux/extlinux.conf
sync
sudo reboot
```
