# v0.2.0 A733 EDID 显示适配

日期：2026-07-14

## 范围

本版本提供已验证的 Radxa Cubie A7Z HDMI 原生模式修复。这是显示策略版本，不是新的 Debian 整卡镜像。

## 已验证结果

- 开发板：Radxa Cubie A7Z。
- 内核基线：原厂 `5.15.147-21-a733`（`l1`）。
- 屏幕：`FLY-HDMI-LCD7`。
- 实际模式：`1024x600@60Hz`。
- 结果：KDE 完整桌面正常显示，无拉伸和裁切。

## 发布资产

- `0001-drm-prefer-edid-native-mode.patch`：可重复应用的 BSP 源码补丁。
- `sun60i-a733-cubie-a7z.dtb`：用于原厂 `l1` 内核的预构建 A7Z DTB。
- `SHA256SUMS`：资产校验和。

## 安装

先备份原厂 DTB，再安装发布的 DTB。该操作不替换 kernel、initramfs 或 Wi-Fi 模块。

```bash
sudo cp -an /usr/lib/linux-image-5.15.147-21-a733/allwinner/sun60i-a733-cubie-a7z.dtb \
  /usr/lib/linux-image-5.15.147-21-a733/allwinner/sun60i-a733-cubie-a7z.dtb.pre-v0.2.0
sudo install -m 0644 sun60i-a733-cubie-a7z.dtb \
  /usr/lib/linux-image-5.15.147-21-a733/allwinner/sun60i-a733-cubie-a7z.dtb
sync
sudo reboot
```

重启后验证：

```bash
sudo journalctl -b -k --no-pager | grep -E 'Configuration mode|drm hdmi mode set'
```

测试屏的预期结果为 `1024x600@60Hz` 和 `drm hdmi mode set: 1024*600`。
